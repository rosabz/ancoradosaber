## Âncora do Saber
Sistema web desenvolvido para auxiliar estudantes na organização e planejamento dos estudos.
## Objetivo Geral
Desenvolver um site de planejamento de estudos online que ajude estudantes a organizar sua rotina, melhorar a produtividade e acompanhar seu desempenho acadêmico.
## Objetivos Específicos
- Criar e organizar cronogramas de estudo  
- Gerenciar tarefas com prazos e prioridades  
- Acompanhar o progresso dos estudos  
- Organizar atividades por matérias  
- Visualizar metas semanais  
- Exibir lembretes importantes  
- Disponibilizar um painel com resumo das atividades  
## Funcionalidades
- Cadastro de usuário  
- Login de usuário  
- Criação de tarefas  
- Listagem de tarefas  
- Visualização de progresso  
- Dashboard com resumo de atividades  
## Tecnologias Utilizadas
- HTML  
- Bootstrap  
## Estrutura do Projeto
- `index.html` → Dashboard  
- `login.html` → Página de login  
- `cadastro.html` → Página de cadastro  
- `tarefas.html` → Lista de tarefas
